About

Project Goal: The goal of this project is to develop an advanced application for processing historical stock data from the Macedonian Stock Exchange. Using a Pipeline and Filter architecture, the system will enable modular and scalable data processing by automating the retrieval, transformation, and cleaning of daily stock data for all listed companies over the past 10 years. Key functionalities include fetching data from external sources, filtering relevant information (e.g., date, opening/closing prices, trading volume, high/low values), and formatting it for storage as structured and clean datasets ready for analysis. The modular pipeline approach ensures flexibility, where each filter performs a specific task, passing its output to the next stage. This design lays the groundwork for future expansions, such as integrating advanced analytics tools or additional features.
